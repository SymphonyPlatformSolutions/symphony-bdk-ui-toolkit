# Toggle

<hr />

Toggle component. Works pretty much like a checkbox.

<Preview withSource="none">
  <Story name="Base">
    <ToggleController />
  </Story>
</Preview>

This component likely most Inputs, it a controlled component, therefore its usual that its wrapped
by a controller, or its state and logic by its parent.

### Controller example

```jsx
const ToggleController = (props) => {
  const [toggled, setToggle] = useState(true);

  return (
    <Toggle
      {...props}
      toggled={toggled}
      onChange={setToggle}
    />
  );
};
```
