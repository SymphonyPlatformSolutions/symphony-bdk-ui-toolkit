# Box

<hr />

**Box** is our main structuring component, wrapping any content with a div with predetermined (but extensible) dimentions. Close to all other components in the Toolkit use Box.

Most props for the Box are a simplified relay of common CSS flex-box positioning (such as direction and padding).

  <ComplexExample />
</Story>

## Main properties

> **An Observation:**
>
> The ```HilighterBox``` component showcased here in the code is simply a styled Box
> with padding, 100% width and background color

```jsx
const HilighterBox = (props) => {
  const { children, style, ...rest } = props;
  return (
    <Box
      {...rest}
      style={{
        width: '100%',
        backgroundColor: 'rgba(53, 56, 62, 0.2)',
        padding: '10px',
        ...style,
      }}
    >{children}
    </Box>
  );
};
```
